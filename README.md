AI-first support platform across 5 e-commerce stores with LLM-driven routing, tool invocation, multi-provider abstraction, and Grafana observability.

- 75% automated query resolution
- CSAT maintained above 90%
- Unified chat, email, form, and voice channels
- ERP, CRM, and courier API integrations

### 2. AI Agentic Product Content Generation
Multi-agent pipeline automating supplier data ingestion, attribute extraction, and product publishing. Scaled catalogue from 12k to 40k+ SKUs.

- 90% reduction in manual product creation
- 300%+ revenue growth contribution
- 60+ attribute sets, 300+ attribute values
- Pinecone RAG for taxonomy classification
